## ✅ 과제 목표

- Flutter의 기본 위젯을 사용해 하나의 화면을 구성해봅니다.
- `Text`, `TextField`, `Button`, `Image`, `ListView` 등 자주 쓰이는 위젯을 실습합니다.
- 상태 관리(`setState`)를 사용해 **입력값 반영** 또는 **버튼 클릭 시 상태 변화**를 구현합니다.
